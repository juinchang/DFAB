Laser Cut

Go to modify > change parameters menu
Set the variable/parameter in the paramater table
Name your measurement(eg, length). Select the unit, and the value you want to set
Start sketch, using the parameters you set for the value respectively
When drawing the side, you can use the collinear to make sure its alige exactly.
Select both front and side, and cut it using the combine tool
Create a mid plane and mirror the two objects
And you get the box
